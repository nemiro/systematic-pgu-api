Виджет авторизации

Виджет авторизации представляет из себя всплывающее модальное окно (в едином стиле для всех порталов) для авторизации через SSO ПГУ МО. Позволяет авторизовывать пользователя через различные данные (связка email-пароль, ЕСИА, соц. сети). Модальное окно всплывает поверх других окон, перекрывая интерфейс. Подробнее..

Виджет профиля

Виджет профиля представляет из себя всплавающий блок с информацией о пользователе, который можно дополнительно использовать, для того, чтобы получать некоторую общедоступную информацию о пользователе, например его Ф.И.О. и аватар.
Так же виджет позволяет выводить ссылки на базовые разделы профиля пользователя на uslugi.mosreg.ru , а так же список последних записей истории пользователя. Подробнее..
